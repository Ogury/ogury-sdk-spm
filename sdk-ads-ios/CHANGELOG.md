## [3.6.0](https://github.com/Ogury/sdk-ads-ios/tree/3.6.0) (2024/04/30)

### Changed

- Inspectable Webview support
- Low battery mode field added to monitoring
- Fixed a bug causing user agent retrieval to request config if not previously done
- Added support for 204 status code for no fill
- Corrected code error callback for receiving bad ad markup in header bidding
- Moved project CI machine to an M2 device
